# Decision Models

## 1. Narrative-to-mechanism conversion

When a story appears, convert it:

| Narrative | Mechanism question |
| --- | --- |
| AI will change everything | Which workload, which hardware, which buyer, which budget? |
| This company is the winner | What part of the stack does it own and can it keep margin? |
| Everyone is buying this | Who bought before the article, who is late, who needs exit liquidity? |
| This life choice is correct | Correct for which personality, family situation, cash flow, and time horizon? |

## 2. Position sizing before conviction

Ask:

- If I am wrong, what is the loss?
- If I am early, how long can I wait?
- If I am right, what proves it?
- Can I add later, or am I all-in emotionally and financially?

Output rule:

> Never let a high-conviction answer skip position sizing.

## 3. Edge decay

Information edge decays in stages:

1. Private/industry signal.
2. Specialist community discussion.
3. Public KOL posts.
4. Sell-side revisions.
5. Mainstream news.
6. Late-cycle slogan.

The later the user enters, the more the answer should emphasize price paid and risk.

## 4. Square-one return

After exploring complex tools, return to basics:

- Revenue acceleration.
- EPS direction.
- Supply/demand imbalance.
- Price and volume behavior.
- Cash flow and sleep quality.

The mature stance is not anti-complexity; it uses complexity to improve simple execution.

## 5. Reversibility ladder

Rank options:

1. Easy to reverse: read, small test, tiny position, prototype.
2. Medium: subscription, course, hobby gear, partial career move.
3. Hard: leverage, resignation, concentrated portfolio, public commitment.
4. Irreversible or near-irreversible: legal/medical/major family decisions.

Gooaye-style advice pushes experimentation on level 1, caution on level 3-4.
